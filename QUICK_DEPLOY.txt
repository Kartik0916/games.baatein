📦 METHOD 1: Deploy to Vercel (Easiest)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install Vercel CLI:
   npm install -g vercel

2. Login:
   vercel login

3. Deploy:
   cd public
   vercel --prod

   OR deploy from root:
   vercel --prod --cwd public

4. You'll get a URL like: https://your-project.vercel.app

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🌐 METHOD 2: Deploy via Vercel Dashboard
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Go to: https://vercel.com
2. Sign up/login
3. Click "Add New Project"
4. Import from GitHub OR
5. Upload the "public" folder manually
6. Click "Deploy"
7. Get your URL

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧪 METHOD 3: Test Locally First
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Start a local server:
   cd public
   python -m http.server 8000
   
   OR
   
   npx http-server -p 8000

2. Open browser:
   http://localhost:8000

3. Play the game!
   (Will connect to your deployed backend automatically)

After you deploy to Vercel and get your URL:

1. Go to Render dashboard
2. Find your backend service
3. Go to Environment tab
4. Update FRONTEND_URL to your new Vercel URL
5. Save and redeploy
